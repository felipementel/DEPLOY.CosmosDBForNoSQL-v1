## Executar o Emulador do CosmosDB

```
docker run \
--publish 8081:8081 \
--publish 10250-10255:10250-10255 \
--name canal-deploy-cosmos-emulator \
--detach \
mcr.microsoft.com/cosmosdb/linux/azure-cosmos-emulator:latest
```
